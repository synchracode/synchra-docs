### `{time <timezone>}`

<Aside type="tip" title="Example usage">
`The current time in Tokyo is {time Asia/Tokyo}`
</Aside>

Returns the current time in the given timezone.

<Aside type="note" title="Example response">
  `16:00`
</Aside>
