### `{user}`

This variable refers to the user mentioned as the first argument when someone uses a command or the user who invoked the command if no user is mentioned.

<Aside type="tip" title="Example usage">
`Hello, {user}! Hope you're having a good day.`
</Aside>

**Usage:**

- If someone types `!greet ErlePerle`, the bot will respond with:
  `Hello, ErlePerle! Hope you're having a good day.`

- If someone just types `!greet`, the bot will respond with:
  `Hello, [their own name]! Hope you're having a good day.`
