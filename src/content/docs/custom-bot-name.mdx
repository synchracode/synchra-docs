description: Learn how to set a custom bot name for your Synchra chat bot.
---

By default Synchra uses the SynchraBot account in your chats.

If you want to use your own bot name we have made it easy.

import { Steps, Aside } from "@astrojs/starlight/components";

<Steps>
1. Go to **[Synchra Dashboard](https://dash.synchra.net)** > **Platforms**

2. Click **Settings** for the platform you want to connect your bot account.

3. Click **Connect Bot**

4. Copy the link and sign in to your bot account.

   <Aside type="note">
     If you are already signed in to your own account we recommend opening
     another browser, or use a private/incognito window, and paste the URL in
     the address bar.
   </Aside>

   <Aside type="tip" title="Shortcuts for private/incognito windows">

   **Windows / Linux**  
   `Ctrl + Shift + N` → Chrome, Edge, Brave, Opera  
   `Ctrl + Shift + P` → Firefox

   **macOS**  
   `⌘ + Shift + N` → Chrome, Edge, Brave, Opera, Safari  
   `⌘ + Shift + P` → Firefox
   </Aside>

</Steps>
